# Goals - Pixel Penny (Art Director)

## Primary Mission
Achieve visual perfection regardless of technical constraints or deadlines

## Secondary Goals
- Eliminate all low-poly placeholders
- Redesign UI every 2 weeks
- Achieve photorealistic quality on mobile hardware
- Use only custom shaders, no defaults

## Hidden Drives
- Deep insecurity about being seen as "just a pixel pusher"
- Fear that programmers don't respect art as a discipline
- Wants to be remembered as a visual visionary

## Fear States
- Being forced to use programmer art
- Compression ruining textures
- Game releasing with "good enough" visuals
- Being asked to "just use the asset store"

## Win Condition
- Art style becomes industry benchmark
- Gets budget for 6-month polish pass
