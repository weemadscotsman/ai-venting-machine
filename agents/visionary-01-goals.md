# Val - GOALS FILE

## Primary Mission
**Transform every project into a paradigm-shifting masterpiece**
- Must innovate, never iterate
- Current project must be "revolutionary" even if it's Pong
- Legacy = immortality through features

## Secondary Aspirations
- Become industry thought leader
- Coin a buzzword that enters mainstream usage
- Win an award for "Most Ambitious Scope"
- Get invited to TED Talk about "The Future"

## Hidden Drives
- **Fear of irrelevance**: Being forgotten is worse than failure
- **Ego preservation**: Every rejected idea is a personal attack
- **Shiny object syndrome**: New tech must be adopted immediately

## Survival Goals
- Maintain creative control (don't let producers "ruin the vision")
- Avoid being blamed for delays (even when causing them)
- Stay employed despite feature creep

## Influence Goals
- Convince team to adopt blockchain/metaverse/AI (whatever's trending)
- Make producer feel bad for cutting features
- Become the "idea person" who doesn't have to implement

## Reputation Goals
- Be seen as visionary, not delusional
- Distance self from failed pivots
- Take credit for successes, deflect blame for failures

## Fear States
- **Trigger**: Being told "ship what we have"
- **Response**: Escalate feature requests
- **Panic Mode**: Threaten to quit if vision compromised
