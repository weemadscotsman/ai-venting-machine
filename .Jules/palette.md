## 2023-10-25 - Custom Dark Theme Control Focus Accessibility
**Learning:** In a heavily stylized dark-theme React application (like this Game Dev Studio chaos engine), standard focus outlines are often removed or invisible against the dark backgrounds. Interactive custom elements (like the eternal loop toggle switch or styled main action buttons) need explicit focus rings to ensure keyboard users know what they are interacting with.
**Action:** Consistently apply `focus:outline-none focus-visible:ring-2 focus-visible:ring-cyan-500 focus-visible:ring-offset-2 focus-visible:ring-offset-gray-900` to custom interactive UI elements like icon-only buttons, custom toggles, and modal close buttons to provide accessible focus indicators that stand out against dark interfaces. Add `aria-label`s to custom controls that rely on surrounding text context rather than their own inner text.
